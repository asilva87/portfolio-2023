Mengenlehreuhr
This is a replication of the famous Mengenlehreuhr located in Berlin.
It functions like so:
Round light represents seconds.
First row, each block represents 5 hours.
Second row, each block represents 1 hour.
Third row, each block represents 5 minutes.
Fourth row, each block represents 1 minute.
You read it by adding the 5 hours blocks, then the 1 hour blocks, following by the 5 hours blocks and the 1 minute blocks.
